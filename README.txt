CUSTOMIZATION:
--------------
To change server information, open index.html and update:

1. Server IP (line 52): Change "connect.rustyamerican.com" to your actual server IP
2. Player count (line 38): Currently shows 0-200 players
3. Wipe schedule (line 44): Currently set to "Weekly" and "Thursday 2PM EST"
4. Map size (line 127): Currently set to "4000"
5. Max group size (line 131): Currently set to "8 Players"
6. Discord link (line 193): Add your Discord invite URL
